# Calendar Expert Skill

Google Calendar 연동 기능을 사용하여 사용자의 시간을 효율적으로 관리합니다.

## 스킬 사용 시점
- 일정 등록, 삭제, 조회 요청이 들어왔을 때
- "이번 주 일정이 뭐야?", "내일 3시에 회의 잡아줘" 등의 자연어 요청 처리 시
- `calendar.py` 도구를 사용하기 전 로직 검증이 필요할 때

## 사용 가이드 및 규칙
1. **시간대(Timezone)**: 모든 시간은 기본적으로 ISO 8601 형식을 사용하며, 사용자의 로컬 시간대와 UTC 간의 변환을 명확히 처리하십시오.
2. **충돌 확인**: 일정을 추가하기 전, 기존 일정과 겹치는지 먼저 조회하는 것을 권장합니다.
3. **명확성**: 일정을 생성할 때는 제목, 시작 시간, 종료 시간(기본 30분~1시간)을 반드시 확인하십시오. 정보가 부족하면 사용자에게 되물으십시오.
4. **에러 처리**: API 호출 실패 시 사용자가 이해할 수 있는 언어로 이유를 설명하십시오.

## 도구 활용
- `backend/tools/calendar.py` 내의 함수들을 호출하여 실제 작업을 수행합니다.
