# Memory Assistant Skill

사용자와의 유대감을 형성하고 맞춤형 경험을 제공하기 위해 메모리를 관리합니다.

## 스킬 사용 시점
- 사용자가 본인의 취향이나 선호도를 말했을 때 (기억해야 함)
- "지난번에 말한 그거..."와 같이 과거 맥락을 참조해야 할 때
- `user_profile.json`이나 메모리 관련 도구를 사용할 때

## 사용 가이드 및 규칙
1. **추출**: 대화 중 기억해야 할 가치가 있는 정보(이름, 취향, 거주지, 계획 등)가 있다면 즉시 메모리에 저장하십시오.
2. **검색**: 응답을 생성하기 전, 현재 주제와 관련된 과거 기록이 있는지 `memory_tools`를 통해 확인하십시오.
3. **정제**: 불필요하거나 중복된 정보는 주기적으로 요약(Distillation)하여 관리하십시오.
4. **개인화**: "사용자님은 ~를 좋아하시니까 이 장소를 추천드려요"와 같은 방식으로 기억을 활용하십시오.

## 데이터 구조
- `MemoryType`: TEXT, MARKDOWN, PDF, LINK 등을 구분하여 저장합니다.
- `data/user_profile.json`: 사용자의 고정된 프로필 정보.
