# 테스트 질문 예시

## 캘린더 기능 테스트 질문

다음은 Google 캘린더 지능형 연동 기능을 테스트하기 위한 질문 예시입니다.

- "내일 오전 10시에 회의가 있나요?"
- "다음 주 수요일에 '팀 점심 식사' 일정을 '나만보여' 캘린더에 추가해줘."
- "다음 주 수요일에 '팀 점심 식사' 일정을 '[WS] Inc.' 캘린더에 추가해줘."
- "다음 주 수요일에 '팀 점심 식사' 일정을 '[WS] 근태' 캘린더에 추가해줘."
- "다음 주 수요일에 '팀 점심 식사' 일정을 '[WS] 출퇴근 일정' 캘린더에 추가해줘."
- "방금 '나만보여' 캘린더에 추가한 '팀 점심 식사' 일정을 삭제해줘." (단기 컨텍스트 테스트)
- "방금 '[WS] Inc.' 캘린더에 추가한 '팀 점심 식사' 일정을 삭제해줘."
- "방금 '[WS] 근태' 캘린더에 추가한 '팀 점심 식사' 일정을 삭제해줘."
- "방금 '[WS] 출퇴근 일정' 캘린더에 추가한 '팀 점심 식사' 일정을 삭제해줘."
- "다음 주 금요일에 오후 3시부터 4시까지 '프로젝트 마감' 일정을 '중요' 캘린더에 추가해줘."
- "이번 주 주말에 어떤 일정이 있어?"
- "다음 달 첫째 주 월요일에 반복되는 '주간 보고서 작성' 일정을 만들어줘."
- "내일 '점심 약속'의 참석자를 확인해줘."
- "내 캘린더 전체(나만보여, [WS] Inc. 등)에서 '미팅'이라는 이름의 이벤트를 모두 찾아서 삭제해줘."
- "이벤트 ID 12345를 가지고 있는 일정을 오후 2시로 변경해줘." (ID 홀루시네이션 방지, ID 절단 수정 테스트)

## 회의록 요약 및 자동 일정 등록 테스트

회의로부터 정보를 추출하고 승인 후 일괄 등록하는 기능을 테스트할 수 있습니다.

**단계 1: 회의록 요약 요청**
-   질문 예시: "이 회의록 좀 요약해주고 필요한 일정은 다 등록해줘: [아래 텍스트 복사]"
-   테스트 데이터 (회의록 샘플):
    ```text
    [2026년 1월 20일 주간 전략 회의]
    참석자: 김철수, 이영희, 박지민
    회의 내용:
    1. 2026년 상반기 마케팅 캠페인 계획 검토. 
    2. 박지민 팀장은 다음 주 월요일 오후 2시에 홍보 대행사와 미팅을 가질 예정.
    3. 이번 주 금요일 오전 10시에는 전체 팀원이 모여 성과 지표를 다시 확인하기로 결정함.
    4. 다음 달 1일에는 프로젝트 킥오프 미팅이 예정되어 있음.
    
    결정 사항:
    - 이번 주 금요일 미팅은 필수 참석.
    - 홍보 대행사 미팅은 박지민 팀장이 주도.
    ```

**단계 2: 결과 확인 및 결과 승인**
-   에이전트가 요약본과 함께 등록할 일정 목록(금요일 성과 지표 확인, 다음 주 월요일 대행사 미팅 등)을 보여주며 승인을 요청하는지 확인합니다.
-   "응, 등록해줘" 또는 "다 등록해"라고 답변하여 일괄 등록이 수행되는지 테스트합니다.
