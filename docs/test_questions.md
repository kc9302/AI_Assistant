# 테스트 질문 예시

## 캘린더 기능 테스트 질문

다음은 Google 캘린더 지능형 연동 기능을 테스트하기 위한 질문 예시입니다.

- "내일 오전 10시에 회의가 있나요?"
- "다음 주 수요일에 '팀 점심 식사' 일정을 추가해줘."
- "방금 추가한 '팀 점심 식사' 일정을 취소해줘." (단기 컨텍스트 테스트)
- "다음 주 금요일에 오후 3시부터 4시까지 '프로젝트 마감' 일정을 '중요' 캘린더에 추가해줘."
- "이번 주 주말에 어떤 일정이 있어?"
- "다음 달 첫째 주 월요일에 반복되는 '주간 보고서 작성' 일정을 만들어줘."
- "내일 '점심 약속'의 참석자를 확인해줘."
- "캘린더에 있는 '미팅'이라는 이름의 이벤트를 모두 삭제해줘."
- "이벤트 ID 12345를 가지고 있는 일정을 오후 2시로 변경해줘." (ID 홀루시네이션 방지, ID 절단 수정 테스트)

## RAG (Retrieval-Augmented Generation) 기능 테스트 질문

다음은 오사카 여행 지침서 RAG 기능을 테스트하기 위한 질문 예시입니다.

- "오사카 가는 비행기 편명하고 시간 알려줘"
- "우리 숙소 주소랑 체크인 시간 확인해줄래?"
- "도톤보리 근처에서 먹을만한 라멘집 추천해줘"
- "여행 2일차 일정 좀 다시 보여줘"
- "교통편 정보는 어디서 찾을 수 있어?"
- "일본 유심 구매는 어떻게 해야 해?"
- "환전은 어디서 하는 게 좋을까?"
- "오사카의 유명한 관광지는 어디야?"
- "현지에서 응급상황 발생 시 어떻게 해야 해?"
